 High-Definition Models
Sven Co-op 3.0 features several updated monster models that are of a higher quality than the original retail versions. These models were created for Sven Co-op by the Half-Life Improvement Team, who we thank for all their hard work over the past few months.

  If you experience slowdown while playing Sven Co-op, you may wish to try uninstalling the High-Definition models.

Un-installing the High-Definition Models

If you experience slowdown while using these models, you can revert to the original low-def models either by:

Selecting "Low-Definition Content" from the Sven Co-op/Options/ Start menu group

Or by: Running the "Content-LowDef.bat" file in your Half-Life\SvenCoop\ directory.

If you later wish to re-install the Sven Co-op High Definition models, just use the corresponding high-definition start menu shortcut or BAT file.
